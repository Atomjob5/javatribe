爪哇部落——程序员的梦工厂

了解我们

  • 爪哇部落是一支由广东金融学院学子自发组织成立的兴趣型计算机技术学习交流团队。 2011年6月，爪哇部落由广东金融学院2009级计科系的几位充满热血的师兄师姐创立诞生，经过6年的发展，已成为一支极具计算机专业性质的团队。爪哇部落立志于向广金学子提供一个计算机专业的学习交流平台，为学院不同专业，不同年级同学提供专业的学习和就业指导。团队秉承着海纳百川的态度，积极热情的展开各种学习交流活动，结合广东金融学院金融的特色，在创新创业的狂潮中，不断进步。

  • 美工设计组

    美工设计组主要从事设计工作，负责管理美工设计组的常规培训工作。由于目前美工设计组人才缺失，应注重培养在美工方面有兴趣的成员，同时，当各开发组需要美工设计时，必须积极配合。同时，及时向研发部部长汇总日常工作情况。

  • Web开发组

    Web开发组包含Web前端和Web后台，负责管理美工设计组的常规培训工作。由于目前美工设计组人才缺失，应注重培养在美工方面有兴趣的成员，同时，当各开发组需要美工设计时，必须积极配合。同时，及时向研发部部长汇总日常工作情况。

  • 移动开发组

    移动开发组主要偏向Android和ISO开发，可涉及后台。负责管理移动开发组的常规培训工作。由于目前移动开发组人才缺失，应注重培养在移动开发方面有兴趣的创业，指导移动开发组组员进行相应技术方向的确定和技术知识学习和交流分享，分配相应实践任务。同时，及时向研发部部长汇总日常工作情况。
